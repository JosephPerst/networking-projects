# Networking Projects

Computer networking implementations including TCP/UDP protocols, routing, and network applications.

## Projects

| File | Description |
|------|-------------|
| `router.py` | Router implementation with routing logic |
| `tcp_client.py` | TCP client implementation |
| `tcp_server.py` | TCP server implementation |
| `udp_client.py` | UDP client implementation |
| `udp_server.py` | UDP server implementation |
| `network_app.py` | Network application layer |
| `switch.py` | Network switch implementation |

## Topics Covered
- **Transport Layer**: TCP and UDP implementations
- **Network Layer**: Routing algorithms
- **Data Link Layer**: Switch functionality
- **Socket Programming**: Client-server architecture

## Tech Stack
- Python 3.8+
- Socket programming
- Threading

## Usage

Start a TCP server:
```bash
python tcp_server.py
```

Connect with a client:
```bash
python tcp_client.py
```
